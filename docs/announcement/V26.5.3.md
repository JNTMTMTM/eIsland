# eIsland V26.5.3 更新日志

> **发布日期：** `2026-05-06`
> **Github 仓库：** `https://github.com/JNTMTMTM/eIsland`

本次版本聚焦于 **闹钟能力完善**、**全展开布局自定义** 与 **Agent 本地工具扩展**。同时补齐多处翻译与交互细节，提升稳定性和整体使用体验。

## 新功能

- 新增 `maxExpand` 闹钟页面，支持闹钟列表管理、创建、编辑、删除、重复规则设置与启停控制。
- 新增闹钟全局监听与通知能力，闹钟触发时可在系统层面进行提醒。
- 新增全展开页面顺序自定义能力，支持按个人习惯调整导航顺序。
- 新增全展开布局「恢复默认」能力，便于一键回退到标准配置。
- 新增 Agent 对本地待办事项与闹钟的操作能力，支持在对话中直接发起相关管理操作。
- 新增用户自定义 Agent LLM API Key 能力，并为 Pro 场景提供清晰标识。

## 体验优化

- 闹钟编辑页改为左右分栏，编辑路径更清晰，创建与修改效率更高。
- 闹钟时间选择器交互与动画持续优化，滚轮/选择体验更顺滑。
- 闹钟创建入口与操作按钮布局进一步收敛，日常使用更顺手。
- 优化全展开布局预览区与导航点视觉表现，状态识别更直观。
- 替换并统一多处矢量图标，减少硬编码样式带来的视觉不一致。
- 调整用户中心快捷导航卡片默认顺序，常用入口触达更快。

## 问题修复

- 修复闹钟相关工具在 Agent 调用链路中无法正常执行的问题。
- 修复可重复创建同一时间闹钟的问题，避免重复提醒。
- 修复全展开界面初始索引异常导致的默认落点错误。
- 修复自定义 API Key 下推理强度文案显示错误的问题。
- 修复 Agent 自定义 API Key 凭据未及时失效的问题。
- 修复多处架构层稳定性问题，提升运行可靠性。

## 文档更新

- 补齐设置快速导航卡片相关翻译。
- 补齐闹钟相关界面翻译。
- 补齐 `maxExpand` 界面翻译。
- 补齐自定义 API Key 相关翻译。
