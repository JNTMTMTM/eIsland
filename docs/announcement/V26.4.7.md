# eIsland V26.4.7 更新公告

> **发布日期：** `2026-04-25`

本次版本重点围绕 **Pro 订阅支付链路**、**订单管理体验** 与 **本地文件搜索能力** 进行了增强，并同步完成多处稳定性与界面优化。

## 新功能

- 新增本地文件查找功能，支持在设置中配置最大检索数量、目录深度与大小写策略。
- 新增用户订单历史界面与「查看订单」入口，可更直观地管理个人支付记录。
- 新增待支付订单状态卡片，支持刷新订单状态、继续支付与快捷查看订单详情。
- 支持支付宝支付，并在支付前自动校验可用支付渠道状态。
- 支持创建订单时进行滑块验证码校验，提升下单流程安全性。
- Pro 价格与权益信息改为服务端动态获取，价格展示与权益描述更易统一维护。
- 用户中心新增 Pro 功能导航卡片，并可展示 Pro 到期时间信息。

## 体验优化

- 优化支付界面整体布局与支付方式图标展示，关键信息识别更清晰。
- 优化用户总览页与 Pro 卡片视觉表现，账户信息层级更明确。
- 优化本地文件搜索交互细节，常用筛选配置更易调整。
- 优化公告 Markdown 渲染样式，阅读体验更统一。

## 问题修复

- 修复集成灵动岛模式下支付界面显示异常的问题。
- 修复一处架构稳定性问题，提升整体运行可靠性。

## 文档更新

- 完善应用内 ICP 与公安备案相关信息展示。
