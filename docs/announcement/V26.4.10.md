# eIsland V26.4.9-V26.4.10 hotfix 更新公告

> **发布日期：** `2026-04-26`

本次版本围绕 **相册能力** 做了完整补强：总览小组件与 MaxExpand 相册页在交互上更加统一，设置项更完整，媒体切换与预览体验更流畅稳定。

## 新功能

- 总览界面新增相册轮播小组件，支持手动上一张 / 下一张与暂停 / 继续轮播。
- 软件设置新增相册配置页，可独立配置总览相册卡片的轮播间隔、轮播顺序、资源筛选与点击行为。
- 新增视频相关配置开关，可按需控制总览卡片中的视频自动播放与静音策略。
- 布局预览区域支持相册卡片效果预览，便于在设置阶段快速确认展示效果。

## 体验优化

- 优化总览相册卡片信息层布局：页码、标题与控制按钮位置重排，视觉层级更清晰。
- 相册卡片的名称、页码与控制按钮改为 hover 渐显，减少静态遮挡并提升聚焦体验。
- 统一相册媒体切换动效：总览卡片与 MaxExpand 预览均支持左右平滑切换动画。
- 调整布局预览中的相册媒体容器尺寸，预览观感更贴近设置场景。
- 删除 Overall 相册卡片小组件预载媒体资源机制。

## 问题修复

- 修复部分场景下视频资源未正常加载的问题，提升视频预览可用性。
- 优化 MaxExpand 相册页媒体加载时机，降低展开动画阶段的卡顿感。
- 修复相册配置与展示状态同步不稳定的问题，配置变更可即时生效。
- 修复 MaxExpand 界面矢量图标渲染失败问题。
