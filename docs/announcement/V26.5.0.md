# eIsland V26.5.0 更新公告

> **发布日期：** `2026-04-30`
> **Github 仓库：** `https://github.com/JNTMTMTM/eIsland`

V26.5.0 是一次覆盖面较大的功能与稳定性更新。本次发布重点聚焦在 **Agent 体系升级、邮箱 IMAP 能力、余额充值链路、系统工具扩展** 与 **整体稳定性修复**。

## 新功能

- Agent 支持多会话创建与切换，并支持删除历史会话，日常多任务协作更顺手。
- Agent 新增本地工具能力扩展，覆盖更多文件操作与系统组件调用场景。
- Agent 支持上传附件与上传 Skill，支持桌面拖拽附件并显示文件类型图标。
- Agent 新增高危操作鉴权与授权弹窗信息增强，关键操作前确认更清晰。
- 新增上下文使用量展示、记忆上下文与 Todo List 任务机制，复杂任务跟踪更直观。
- 新增模型 TraceID 追溯与问题反馈入口，排查与上报路径更完整。
- 邮箱模块支持 IMAP 多邮箱配置、收件缓存、邮件主要内容展示与 HTML 正文渲染。
- 新增 IMAP 配置指引与文档跳转入口，首次配置流程更易上手。
- 用户中心新增余额展示与余额充值界面，并完成充值后端联调。
- 订单界面展示更多业务参数，支付相关信息更完整。
- 新增系统组件操作工具与窗口管理相关 Agent 工具，桌面操作能力进一步增强。
- 新增电子木鱼小组件与累计功德计数，同时补充图标与音效资源。
- 支持灵动岛壁纸同步系统壁纸，并新增同步清除策略。

## 体验优化

- 重构并细分 Agent 对话界面与类型结构，界面组织与后续维护性同步提升。
- 优化 Agent 输出区域在长上下文下的渲染性能，降低持续输出过程中的卡顿感。
- 优化代码块、表格、链接展示与时间线排版，可读性与信息密度平衡更好。
- 优化侧边栏展开收起交互、历史会话边框动效与相关图标适配策略。
- 优化高危工具授权弹窗布局与提示信息展示，操作决策成本更低。
- 优化附件上传按钮、附件卡片与拖拽上传性能，上传反馈更及时。
- 优化邮箱收件箱、侧边栏与邮件阅读容器样式，阅读体验更聚焦。
- 优化用户余额卡片与相关导航布局，账户信息展示更清晰。

## 问题修复

- 修复 Agent 流式传输在部分场景失效的问题，提升生成链路稳定性。
- 修复切换界面后 SSE 状态丢失问题，减少中途断流情况。
- 修复历史会话总结错误、会话顺序更新过快等问题，历史记录行为更稳定。
- 修复 Todo List 在时间线中的显示与时序问题，任务过程展示更一致。
- 修复 JSON 异常格式未解包、分片代码块判定错误等输出稳定性问题。
- 修复高危工具授权弹窗出现在错误会话窗口的问题。
- 修复技能拖拽框失效、文本选中冲突与控件可选中异常等交互问题。
- 修复邮箱设置导航错误、链接打开与相关配置流程中的可用性问题。
- 修复多处架构设计与注释规范问题，并完成 `eqeqeq` 规则统一整改。
