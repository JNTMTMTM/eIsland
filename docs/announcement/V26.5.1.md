# eIsland V26.5.1 更新日志

> **发布日期：** `2026-05-03`
> **Github 仓库：** `https://github.com/JNTMTMTM/eIsland`

V26.5.1 是一次以稳定性与可用性为重点的补丁更新，围绕 Agent 交互链路、工具调用体验、邮箱能力和账户相关流程做了持续完善，同时修复了若干影响实际使用的问题。

## 新功能

- 新增两个 Agent r1pxc & edoc。
- 新增 Agent 快照模式 默认使用 alt + p 唤起。
- Agent 支持多会话创建、切换与历史会话删除，复杂任务可按会话分组管理。
- Agent 新增更多本地与系统工具能力，覆盖窗口操作、系统组件调用与文件工具扩展。
- Agent 增加附件上传与桌面拖拽上传能力，并支持 Skill 上传。
- Agent 新增高危操作鉴权机制与授权目的展示，关键操作前确认信息更完整。
- Agent 新增 TraceID 追溯与一键问题上报入口，问题定位与反馈路径更顺畅。
- Agent 引入 Todo List 与记忆上下文能力，并支持上下文使用量展示。
- 邮箱模块支持 IMAP 多邮箱配置、收件缓存、邮件主要内容展示与 HTML 正文渲染。
- 新增余额展示与余额充值流程，并补充订单信息字段展示。

## 体验优化

- 优化 Agent 输出区域性能与长上下文渲染表现，减少连续输出时的卡顿。
- 优化 Agent 授权弹窗布局与交互提示，授权决策更清晰。
- 优化附件上传按钮、附件卡片样式与拖拽上传性能，上传反馈更及时。
- 优化邮箱界面样式、导航切换与阅读区域布局，邮件阅读更聚焦。
- 优化 Agent 时间线展示、代码块样式与链接显示细节，信息阅读更直观。
- 优化用户余额相关卡片布局与入口组织，账户信息查看更清楚。

## 问题修复

- 修复 Agent 流式传输在部分场景失效的问题。
- 修复切换界面后 SSE 状态丢失导致中断的问题。
- 修复历史会话摘要、会话顺序更新频率与本地存储截断等稳定性问题。
- 修复 Todo List 与思考过程在时间线中的时序与渲染问题。
- 修复异常 JSON 输出未正确解包、分片代码块判定错误等解析问题。
- 修复高危工具授权弹窗出现在错误会话窗口的问题。
- 修复 Skill 拖拽框失效、文本选中冲突与若干交互细节问题。
- 修复邮箱设置导航错误与部分邮件配置流程可用性问题。
- 修复打包后 `AIbackground.html` 无法加载的问题，确保语音输入背景页在发布版可正常打开。
- 修复快照模式访问 Agent 时 TraceID 未正常显示的问题。
