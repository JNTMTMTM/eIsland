# eIsland V26.4.8 更新公告

> **发布日期：** `2026-04-26`

本次版本重点完善了 **静态资源节点选择能力** 与 **多节点访问稳定性**，并同步优化了设置文案与接口回退逻辑，让不同网络环境下的资源加载更可控、更稳定。

## 新功能

- 新增天气预警功能，PRO 用户可在天气设置中选择开启。
- 新增和风天气 API 接口，PRO 用户可在天气设置中切换。
- 新增静态资源节点选择能力，可在设置中查看并管理资源拉取节点。
- 支持按账号能力使用不同节点：PRO 用户可选择 `R2 / COS / OSS`，普通用户默认使用 `R2`。
- 资源访问链路支持基于节点偏好返回对应地址，用于头像、壁纸与反馈附件等资源场景。

## 体验优化

- 网络设置页新增节点能力提示与 PRO 标识，功能边界更清晰。
- 节点相关文案拆分为网络模块独立翻译键，中英文提示一致性更好。
- 统一节点默认策略：所有用户默认 `R2`，减少首次使用配置成本。

## 问题修复

- 优化多节点回退逻辑，在目标节点不可用时自动回退 `R2`，降低资源加载失败概率。
- 修复部分天气接口优先级流程中的循环实现问题，提升代码规范一致性与可维护性。
- 修复若干文档与脚本规范项（注释头、函数说明）导致的 lint 报警。

## 文档更新

- 新增并补齐本版本相关公告与多语言说明项，便于后续发布与追踪。
