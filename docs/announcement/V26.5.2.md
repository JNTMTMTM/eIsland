# eIsland V26.5.2 更新日志

> **发布日期：** `2026-05-05`
> **Github 仓库：** `https://github.com/JNTMTMTM/eIsland`

---

本次版本围绕本地 AI 能力、MaxExpand 生产力体验与交互可配置性进行了集中升级，同时修复了多项影响稳定性与显示一致性的问题。

## 新功能

- 新增本地 `Ollama` 接入能力，支持连接本地模型并在客户端内完成对话流程。
- 新增 `Ollama` 本地模型识别与端口探测能力，降低本地接入配置门槛。
- 新增 MaxExpand 备忘录功能，支持多条备忘录管理、快速编辑与查看。
- 新增备忘录书签能力，并支持从语音识别结果快速添加到备忘录。
- 新增空闲态点击展开模式：可设置为悬停不自动展开，点击后再展开。
- 新增本地软件相关工具能力，支持读取已安装程序信息并用于后续能力扩展。

## 体验优化

- 快照模式支持受限 Markdown 字体样式显示，阅读重点信息更直观。
- 新增并整合动画设置页，支持在设置中统一管理动画相关开关。
- 支持调整灵动岛动画速度，状态切换节奏可按个人偏好配置。
- 新增 Expand 与 MaxExpand 页面切换动画开关，交互过渡更清晰。
- 优化设置页与交互行为相关文案及多语言提示，配置项理解更直接。
- 优化备忘录界面颜色自适应与图标表现，浅色/深色主题下可读性更好。

## 问题修复

- 修复多处架构设计问题，提升整体稳定性与可维护性。
- 修复 Expand / MaxExpand 状态无法切换到语音识别模式的问题。
- 修复 `Ollama` 流式输出异常、输出被回退以及 JSON 解构异常等问题。
- 修复注入提示词导致思考输出过长的问题，响应节奏更可控。
- 修复上下文下拉框禁用样式丢失问题。
- 修复相册总览页右下角删除按钮图标渲染异常。

## 文档更新

- 补齐相关注释与说明文档，提升代码可读性与后续协作效率。
