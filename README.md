# eIsland

Windows 灵动岛桌面应用，基于 Electron + React + TypeScript 构建。

支持实时显示天气、歌词、计时器、系统工具快捷操作等功能。

> 许可证说明：本软件开源协议基于 GPLv3，并在其基础上进行了修改与补充（详见 `LICENSE`）。
